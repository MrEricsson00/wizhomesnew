# Wiz Homes - Real Estate Platform

A modern real estate booking platform built with React, TypeScript, and Firebase.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Deploy to Vercel

1. Push your code to a GitHub repository
2. Go to [Vercel](https://vercel.com) and import the repository
3. Vercel will auto-detect the settings from `vercel.json`
4. Click "Deploy"

**Note:** No additional environment variables are required as Firebase config is included in the code.
